My Website

I feel that I have done well on making the media queries for each device - mobile, tablet, and desktop. I made sure that the images several screen widths, so that they won't be overflowing the box or skewed. I also think I like how content is divided into boxes with a shadow when a device or tablet mode is used, and then it won't show any shadow on mobile. I believe that I did well at designing an logo that suits the content of my website and fits in the navigation bar.

There are many factors that I want to improve my website, but there are two of them I feel are much necessary to be fixed. I suspect that too many pictures will be annoying when viewing them on mobile because viewers would have to scroll the screen too much. Maybe I should put all images of artists and their artworks into one slideshow somewhere on the History tag. Another one, the navigation tabs on mobile should be merged into one bar. I planned to use w3schools' Mobile Navigation Menu, but I dropped this plan because it was difficult to keep the tabs arranged horizontally when the website is viewed on a desktop or tablet. Also, I already designed how tags should be arranged before I got the idea of using Mobile Navigation Menu, so I found it tough to redesign how it should look.
